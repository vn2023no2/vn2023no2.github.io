# Log vs Metric


## Log
- Show nhiều giá trị theo timeline
- Tool lưu trữ log: Elasticsearch


## Metric
- Chỉ show 1 giá trị trên prometheus
- Hàm rate trong prometheus 
- Tool lưu trữ metric: Prometheus
